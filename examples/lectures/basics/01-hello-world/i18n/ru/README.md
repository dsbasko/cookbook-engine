# Привет, мир

Первый урок нужен, чтобы прогнать весь конвейер рендеринга движка: заголовки
формируют оглавление, блоки кода проходят через Shiki, а GitHub-алерты
превращаются в callout-ы. Содержание не важно — важно, что каждая возможность
Markdown переживает сборку.

## Блок кода

Блок ниже должен получить подсветку синтаксиса от Shiki на странице:

```ts
export function greet(name: string): string {
  return `Привет, ${name}!`;
}

console.log(greet('мир'));
```

## Callout

> [!NOTE]
> Этот алерт проверяет плагин `remark-github-blockquote-alert`. Он должен
> отрендериться как стилизованный callout, а не как обычная цитата.

> [!WARNING]
> Второй алерт с другой важностью, чтобы стилизация callout-ов проверилась более
> чем для одного варианта.
