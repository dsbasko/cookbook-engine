# Второй урок

Этот урок намеренно существует только на русском языке. Английская страница
`/en/basics/02-second-lesson` должна отрендериться с fallback-баннером и русским
содержимым — так проверяется, что движок корректно деградирует, когда перевод
отсутствует, и помечает страницу `noindex`.

## Зачем это нужно

Покрытие fallback-сценария — часть smoke-теста: без него регрессия в логике
`getLessonContent` / `TranslationBanner` прошла бы незамеченной.

## Ещё один блок кода

```go
package main

import "fmt"

func main() {
    fmt.Println("fallback works")
}
```

Навигация ведёт назад к [первому уроку](../../../../basics/01-hello-world/i18n/ru/README.md)
и вперёд — в модуль «Продвинутое».
