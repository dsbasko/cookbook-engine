# Глубокое погружение

В модуле «Продвинутое» один урок. Его единственная задача — дать sidebar и
главной странице второй модуль и заставить кросс-модульную навигацию prev/next
переходить через границу модуля: из последнего урока `basics` сюда.

## Кросс-модульная навигация

Ссылка «назад» на этой странице должна вести ко второму уроку модуля `basics` —
это доказывает, что `getPrevLesson` ходит между модулями, а не только внутри
одного.

## Таблица

| Возможность     | Что проверяет           |
| --------------- | ----------------------- |
| Оглавление      | заголовки `##`          |
| Подсветка кода  | блоки в ```             |
| Callout-ы       | алерты `[!NOTE]`        |
| i18n fallback   | урок только на русском  |

## Завершение

```bash
pnpm -C examples/web build
```

Запуск команды выше — smoke-тест всего движка.
